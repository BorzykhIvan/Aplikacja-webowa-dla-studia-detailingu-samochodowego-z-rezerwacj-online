Web-serwis umożliwiający rezerwację usług detailingu samochodowego online. Projekt integruje się z Google Calendar w celu zarządzania harmonogramem wizyt oraz wykorzystuje Google Firebase do obsługi bazy danych i uwierzytelniania użytkowników.

Główne funkcjonalności:

Przegląd dostępnych usług detailingu.

Rezerwacja terminu wizyty przez stronę internetową.

Synchronizacja rezerwacji z Google Calendar.

Bezpieczna obsługa danych klientów przy użyciu Google Firebase.

Technologie
Frontend: HTML5, CSS3, JavaScript

Backend: Node.js / Express.js

Baza danych: Google Firebase (Firestore)

Integracje: Google Calendar API, Firebase Authentication

Hosting: Firebase Hosting (lub inny)

Funkcje
Formularz rezerwacji z wyborem usługi, daty i godziny.

Autoryzacja użytkownika (opcjonalnie: logowanie za pomocą Google).

Powiadomienia e-mail o dokonanej rezerwacji (opcjonalnie).

Panel administracyjny do podglądu i zarządzania rezerwacjami (opcjonalnie).
